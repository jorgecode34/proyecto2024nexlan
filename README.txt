Descarga del proyecto desde GitHub:
1. Inicialmente, dentro de la pestaña del repositorio de GitHub de nuestro proyecto, en la sección "Código", se verá un botón verde llamado "Código". Al presionarlo, su última opción del menú desplegable indica descargar zip, lo que descargará la carpeta comprimida con todos los archivos necesarios.

Descompresión de la carpeta:
2. Se debe descomprimir la carpeta zip llamada "proyecto2024nexlan-main.zip"

Movimiento de la carpeta al directorio de XAMPP:
3. Mover la carpeta descomprimida a C:\xampp\htdocs

Configuración de la base de datos:
4. Para configurar la base de datos, se debe debe abrir XAMPP y estar ejecutando Apache y MySQL. Luego, en su navegador de preferencia, tiene que dirigirse http://localhost/phpmyadmin. Luego, haga click en el boton con el simbolo de una base de datos que dice "Nueva". Seleccione la pestaña "Importar". Seleccione el archivo base.sql que se encuentra en la carpeta descomprimida y haga clic en "Importar" para cargar la estructura y los datos necesarios.

Visualización del proyecto:
5. Para acceder al proyecto, se debe abrir su navegador web de preferencia y escribir la url http://localhost/proyecto2024nexlan-main
